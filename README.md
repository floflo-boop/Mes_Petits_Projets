# Objectif de ce repo : 

Ce repo à pout but de contenir des projets réalisés par mes soins. 

Ces projets ont pour but de me faire pratiquer différent langages de programmations afin de me familiariser avec ces derniers. Ainsi ils seront de natures variées, avec des implications variées. 

<br>

# Fonctionnement du repo : 

Chaque projet sera consigné dans un dossier. 

Dans ce dossier, il y aura : 

- Un dossier application qui va contenir les scripts commentés ainsi qu'un fichier requirements.txt afin de copier facilement l'environement du projet.
- Un dossier documentation qui va contenir lui un fichier explicatif du projet : pourquoi ce projet ? Quel but à ce projet ? Quelles contrainte à ce projet ? Quelle solutions et pourquoi ? Et un second fichier plutôt manuel qui expliquera tout simplement comment se servir du projet.
